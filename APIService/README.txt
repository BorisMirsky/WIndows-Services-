Основная идея:
При запуске летит по адресу (АПИ) с реквестом, получает респонс и пишет его в файл с датой.
Одна дата - одна запись.

При запуске проверяет:
- какая дата сегодня
- есть ли запись за предыдущие дни. Если нет то восполняет.
- пишет за сегодня.
